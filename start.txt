1) Создаем директорию проекта
2) Создаем виртуальное окружение и активируем
3) Создаем файл ruquirements.txt, куда указываем используемые библиотеки.
4) Устанавливаем библиотеки при помощи  pip install -r requirements.txt
5) В Pycharm открываем настройки: Preferences -> Project -> Python Intrepreter и выбираете версию Python из окружения
6) Проверить, что окружение активировано в терминале Pycharm
7) Создаем Django - проект командой: django-admin startproject название_проекта . (ставим точку обязательно)
8) Создаем нужные приложения командой python3 manage.py startapp название_приложения
